| Boundary | Description |

| Kafka broker connection | SDK → external Kafka cluster; SASL credentials cross this boundary |
| SignalBatch.Signals | Internal signal data marshalled to JSON and published externally |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-06 | Information Disclosure | SASL credentials in Config.SASL | mitigate | Password field sourced from secrets.Store at runtime; Config never written to logs (no zap.Any(cfg)); connector logs broker addresses only |
| T-02-07 | Tampering | TLS for Kafka transport | mitigate | TLS built via connector.NewTLSConfig which enforces MinVersion=TLS13 and InsecureSkipVerify=false |
| T-02-08 | Spoofing | Kafka producer identity | mitigate | SASL authentication (PLAIN or SCRAM) required when cfg.SASL.Mechanism non-empty; framework validates mechanism string and returns error on unknown values |
| T-02-SC | Tampering | npm/pip/cargo installs | mitigate | franz-go is a Go module fetched via go get; slopcheck + verify go.sum integrity before use |
</threat_model>
