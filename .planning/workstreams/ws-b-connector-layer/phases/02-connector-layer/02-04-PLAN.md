| T-02-09 | Information Disclosure | HEC token in Authorization header | mitigate | Token sourced from secrets.Store at runtime; never logged (log endpoint URL only, not auth header) |
| T-02-10 | Tampering | TLS enforcement for HEC | mitigate | HTTP client Transport.TLSClientConfig built via connector.NewTLSConfig (TLS 1.3 minimum, InsecureSkipVerify=false) |
| T-02-11 | Tampering | HEC response parsing | mitigate | Response body unmarshal into fixed struct; JSON decode error treated as delivery failure; no dynamic code execution |
