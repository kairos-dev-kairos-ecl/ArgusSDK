  (Buffer already imports this: "github.com/kairos-dev-kairos-ecl/ArgusSDK/internal/connector")

Proto serialization for SignalBatch records:
  Use encoding/gob rather than protobuf for buffer records (SignalBatch is an internal Go struct, not a proto-generated type). The design doc says "serialised SignalBatch bytes" — use gob.Encode since SignalBatch has no proto methods.
  File format per design doc: 4-byte big-endian uint32 length (of gob-encoded bytes) + gob-encoded SignalBatch bytes.

Backoff formula:
  attempt 0: BackoffBase
  attempt N: min(BackoffBase * 2^N, BackoffMax) + rand.Int63n(BackoffJitter.Nanoseconds())
  Use math/rand (not crypto/rand) for jitter.

WAL segment naming:
  <Dir>/wal-<unix_seconds>.seg — one active segment per buffer lifetime.
  On close or size limit: rotate (create new segment file).
  Confirmed records: overwrite first byte of record header with 0xFF as a "consumed" marker — on read, skip records with 0xFF marker. Simple and correct for single-writer single-reader use.

Atomic counters: use sync/atomic for buffered_batches, buffered_bytes, dropped_batches.

    Define openSegment(path string) (*walSegment, error): open or create file with os.OpenFile(path, os.O_CREATE|os.O_RDWR|os.O_APPEND, 0600).

    Define func appendRecord(seg *walSegment, batch *connector.SignalBatch) (int, error):
      Acquire seg.mu.
      gob.Encode batch into a bytes.Buffer.
      Encode: write 4-byte big-endian uint32 (length of gob bytes) then gob bytes.
      Append using seg.file.Write. Return bytes written, error.
      After write, call seg.file.Sync().

    Define func streamRecords(path string, fn func(offset int64, batch *connector.SignalBatch) error) error:
      Open file for reading. Read records sequentially:
        Read 4-byte length prefix (binary.BigEndian.Uint32).
        Read that many bytes.
        If first byte of the 4-byte prefix is 0xFF, skip the record (consumed marker).
        Otherwise gob.Decode into a connector.SignalBatch. Call fn(currentOffset, &batch).
        On fn error: return it (stops streaming).
      Return nil at EOF.

    Define func markConsumed(path string, offset int64) error:
      os.OpenFile(path, os.O_RDWR, 0600). Seek to offset. Write single byte 0xFF. Close. Return error.

    Define func segmentPath(dir string) string: return filepath.Join(dir, fmt.Sprintf("wal-%d.seg", time.Now().Unix())).

    Define func segmentSizeBytes(path string) (int64, error): os.Stat(path).Size().

    Imports: "encoding/binary", "encoding/gob", "fmt", "os", "path/filepath", "sync", "time", "github.com/kairos-dev-kairos-ecl/ArgusSDK/internal/connector".

  <action>
    Extend internal/buffer/buffer.go. Keep existing Config, DefaultConfig, New unchanged.

    Add to Buffer struct:
      seg        *walSegment       // active segment
      segPath    string
      segMu      sync.Mutex
      closed     atomic.Bool       // set in Close
      countBatches atomic.Int64
      countBytes   atomic.Int64
      countDropped atomic.Int64

    Write(batch *connector.SignalBatch) error:
      If closed: return fmt.Errorf("buffer is closed").
      Lock segMu. If seg nil: create dir (os.MkdirAll), open new segment at segmentPath(cfg.Dir). Unlock.
      Call appendRecord(seg, batch). On error: return error.
      Check segment size; if > cfg.MaxSizeMB*1024*1024: rotate segment (open new, close old).
      Increment countBatches by 1, countBytes by written bytes.

    Start(ctx, drain): goroutine running drainLoop.

    drainLoop(ctx, drain):
      ticker := time.NewTicker(cfg.FlushInterval).
      backoffAttempt := 0.
      On ticker.C: call drainOnce(ctx, drain, &backoffAttempt).
      On ctx.Done: return.

    drainOnce(ctx, drain, attempt *int):
      If segPath == "": return (nothing buffered).
      Call streamRecords(segPath, func(offset int64, batch *connector.SignalBatch) error:
        err := drain(ctx, batch).
        If err == nil: markConsumed(segPath, offset); return nil.
        If err != nil: increment *attempt; wait = min(BackoffBase * (1<<*attempt), BackoffMax) + jitter; sleep wait; return err.
      ).

    Flush(ctx, drain): call drainOnce synchronously until all records consumed or ctx cancelled.

    Close(): set closed = true. If seg non-nil: call seg.file.Sync() then seg.file.Close(). Return nil.

    Stats(): return map[string]int64{"buffered_batches": countBatches.Load(), "buffered_bytes": countBytes.Load(), "dropped_batches": countDropped.Load()}.

    Write tests in internal/buffer/buffer_test.go (package buffer_test). Use os.MkdirTemp for Dir. Ensure all temp dirs are cleaned with t.Cleanup(func(){os.RemoveAll(dir)}).
