<objective>
Create the connector factory that maps a neutral, agent-independent description of an output
(Type, Endpoint, OCSF, TLS, Auth, Extra) to a constructed, registered connector.Connector, plus
a crypto/rand BatchID generator the agent ingest loop will use. This is the foundational piece
SC-2/SC-3 wiring depends on.

Purpose: SC-1 — `agent.start()` cannot build a ConnectorRegistry without a Type→Connector map.
The factory MUST live in `internal/connector` and MUST NOT import `internal/agent` (that would
create the `agent ↔ connector` import cycle forbidden by locked decision 1). The agent passes a
neutral input struct defined here.

Output: `internal/connector/factory.go`, `internal/connector/ulid.go`, and their unit tests.
</objective>
