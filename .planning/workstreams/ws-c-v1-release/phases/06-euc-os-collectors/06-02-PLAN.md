    newOSCollector(cfg Config) OSCollector returns a windowsCollector holding cfg, a logger, the ETW

    Start(ctx, out):
      1. Create a real-time session with a DETERMINISTIC name (e.g. "ArgusEUC"); stop-if-exists first
         so a stale session from a prior crash is cleared (Runtime State Inventory). On failure to
         create the session: log + return nil (degrade).
      2. EnableProvider for Microsoft-Windows-Kernel-Network (etw.MustParseProvider). REJECT
         NDIS-PacketCapture. If enable fails: log + stop session + return nil.
      3. NewConsumer(ctx).FromSessions(session); set EventCallback to extract remote host/addr,
         local port, and PID from the connect event; if matchHost(remoteHost, cfg.AIEndpoints) OR
         matchPort(localPort, cfg.LocalInferencePorts) (helpers from netcommon.go / 06-00), build
         Observation {ConnectedHost, LocalPort, IsLocal:(matched by local port), ProcessName:
         best-effort from the event PID resolved to its single process, Username: best-effort (empty
         acceptable — NO enumeration)} and send on out respecting ctx.Done().
      4. cons.Start(); on ERROR_ACCESS_DENIED (5) or any consumer-start failure: log + stop session +
         return nil (degrade, Pitfall 3).
      5. Spawn the gopsutil local-inference path (needs NO elevation, runs even when ETW degraded): a
         ticker goroutine calling gnet.ConnectionsWithContext(ctx, "tcp"), matching own-user
         listening/established sockets against cfg.LocalInferencePorts (via the netcommon local-
         inference-port helper), emitting Observation{IsLocal:true, LocalPort, ConnectedHost:
         loopback host} on out, de-duping.
      Always return nil from Start (degrade on any capture-init failure).
